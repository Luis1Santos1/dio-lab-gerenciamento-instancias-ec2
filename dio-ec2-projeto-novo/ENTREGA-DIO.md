# Entrega do desafio na DIO

## Nome do projeto

```text
dio-lab-gerenciamento-instancias-ec2
```

## Descrição para enviar

Desenvolvi um repositório de estudos sobre gerenciamento de instâncias Amazon EC2, documentando desde a criação e configuração inicial até o acesso, monitoramento, gerenciamento do ciclo de vida e encerramento dos recursos. O projeto também reúne boas práticas de segurança, cuidados com custos, comandos úteis da AWS CLI, um script de user data para instalação do NGINX e os principais aprendizados obtidos durante o laboratório.
